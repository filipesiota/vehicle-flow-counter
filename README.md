# vehicle-flow-counter

Contador de fluxo veicular em vídeo (CustomTkinter + OpenCV). Documentação completa virá nas fases finais.

## Requisitos

- Python 3.12
- [uv](https://docs.astral.sh/uv/)
- **Tkinter**: no Debian/Ubuntu/WSL instale `python3-tk` (necessário para CustomTkinter).

## Executar

```bash
uv venv
uv sync
uv run python -m vehicle_flow_counter
```

Alternativa: comando `vehicle-flow-counter` após `uv sync`.
